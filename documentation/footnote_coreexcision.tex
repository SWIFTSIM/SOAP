\paragraph{$^{$FOOTNOTE_NUMBER$}$Core excised quantities}\label{footnote:$FOOTNOTE_NUMBER$} exclude the inner region of the halo when computing the quantity.
They are only calculated for 500 crit. Any core excised calculation only uses the particles
for which

\begin{equation}
     0.15 R_{500c} \leq \mathbf{r} \leq R_{500c}
\end{equation}
